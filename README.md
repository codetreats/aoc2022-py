# aoc2022
Advent Of Code 2022

## Language

Python

## Usage

```
./run.sh [DAY]
```

e.g.

```
./run.sh 1 # runs day 01
```

## Tests

Run all tests:

```
./test.sh
```

## Hints

The puzzle input of each day is given to the `convert` - function of the day.
The result of the `convert` - function is available as self.data.
